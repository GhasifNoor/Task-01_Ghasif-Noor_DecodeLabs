##  About the Project

This portfolio website was built as part of the **DecodeLabs Industrial Training Kit — Project 1: Responsive Frontend Interface**. The goal was to master the fundamentals of frontend development by building a fully responsive, mobile-first website using only HTML, CSS, and JavaScript — without relying on any frameworks like Bootstrap or Tailwind.

The site showcases my skills, projects, and contact information, and is designed to work seamlessly across all screen sizes.

---

##  Features

-  **Fully Responsive** — works on mobile, tablet, and desktop
-  **Hamburger Menu** — collapsible navigation for small screens
-  **DecodeLabs 2025 Color Palette** — Mocha Mousse, Ethereal Blue, Moonlit Grey
-  **Working Contact Form** — powered by Formspree, sends directly to Gmail
-  **Scroll Animations** — sections fade in as you scroll
-  **Active Nav Highlighting** — current section highlighted in navbar
-  **Deployed on GitHub Pages** — accessible on any device anywhere

---

##  Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and semantic markup |
| CSS3 | Styling, layout (Grid & Flexbox), responsive design |
| JavaScript | Interactivity, animations, form handling |
| Formspree | Contact form backend (no server needed) |
| GitHub Pages | Free hosting and deployment |
| Google Fonts | Montserrat + Roboto typography |

> No frameworks. No libraries. Pure fundamentals — as required by DecodeLabs.
